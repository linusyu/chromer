问题

如果计算机上的分层服务提供程序 (LSP) 出现问题而导致 Chrome 浏览器无法载入请求的网页，就会出现此错误。具体而言，LSP 报告的写入套接字的数据比请求的数据更多。如果计算机上有恶意软件，或者计算机上的下载加速器使用的是有问题的 LSP，则可能出现这种情况。

解决方法

检查计算机上是否有恶意软件

Google 网页搜索帮助中心提供了有关如何从计算机中删除恶意软件的说明。

停用 LSP

如果您已确定计算机中没有恶意软件，但该问题仍然存在，请使用 Windows Defender 的 Software Explorer 或其他第三方实用程序停用计算机上安装的 LSP。Microsoft 的支持网站提供了关于使用 Windows Defender 的 Software Explorer 的详情。
