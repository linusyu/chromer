问题

如果您的 Windows 安装文件损坏了，系统就会显示此错误。

解决方案

请检查受保护系统文件是否存在问题。

Windows XP：

  1. 转至开始菜单 > 运行。
  2. 在文本字段中输入以下内容：cmd.exe。
  3. 点击确定。
  4. 在打开的命令提示符窗口中，输入以下内容：sfc /scannow（请确保 sfc 与 /scannow 之间有一个空格）。该命令会启动 Microsoft 实用程序，扫描和修复计算机上受保护系统文件存在的问题。

Windows Vista/Windows 7：

  1. 转至开始菜单 > 运行。
  2. 在文本字段中输入以下内容：cmd.exe。
  3. 当您在搜索结果中看到 cmd.exe 后，请右键点击相关程序并选择以管理员身份运行。
  4. 在显示的确认窗口中点击是。
  5. 在打开的命令提示符窗口中，输入以下内容：sfc /scannow（请确保 sfc 与 /scannow 之间有一个空格）。该命令会启动 Microsoft 实用程序，扫描和修复计算机上受保护系统文件存在的问题。

Windows 8：

  1. 转到搜索。
  2. 点击应用。
  3. 在搜索框中输入：cmd.exe。
  4. 在搜索结果部分中右键点击“命令提示符”。
  5. 点击以管理员身份运行。
  6. 在显示的确认窗口中点击是。
  7. 在打开的命令提示符窗口中，输入以下内容：sfc /scannow（请确保 sfc 与 /scannow 之间有一个空格）。该命令会启动 Microsoft 实用程序，扫描和修复计算机上受保护系统文件存在的问题。
