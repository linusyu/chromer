当网页未正确显示时，就会出现“糟糕！”这一错误。在大多数情况下，您只需重新加载网页即可纠正该错误。如果您不断收到该错误，请尝试按以下建议操作。

在计算机上

如果您是在计算机上使用 Chrome，并且经常看到“糟糕！”这一错误，请尝试使用下面的问题排查选项。

某些防病毒或防火墙应用可能会阻止 Google Chrome 正常运行。要检查您是否属于这种情况，请按以下步骤操作：

检查防病毒和防火墙应用

  1. 退出 Chrome。
  2. 暂时关闭所有防病毒和防火墙应用。
  3. 重新打开 Chrome。如果您没有看到“糟糕！”这一错误，则说明可能是这些应用在干扰或阻止 Chrome 运行。
  4. 在这些应用的设置中，将 Google Chrome 添加为例外（具体流程可能会因应用而异）。
  5. 仔细检查这些应用是否为最新版本。
  6. 重新开启这些应用。

注意：用户已举报下列应用会导致 Chrome 出现“糟糕！”问题：

  • Symantec Endpoint Protection
  • Spyware Doctor
  • Comodo firewall
  • McAfee Enterprise
  • McAfee Virus Scan

检查扩展程序

尝试逐个停用扩展程序，看看问题是不是由某个扩展程序引起的。具体操作步骤如下：

  1. 在浏览器窗口的右上角，点击 Chrome 菜单 。
  2. 依次选择更多工具 > 扩展程序。
  3. 对于您要测试的各个扩展程序，取消选中“已启用”。
  4. 刷新显示“糟糕！”的网页，看看是否还会出现该错误。

您可以随时重新启用这些扩展程序。您可能需要刷新该页面才能使用已重新启用的扩展程序。详细了解如何管理扩展程序。

检查恶意软件

您的计算机上可能安装了干扰 Chrome 的程序。请检查是否存在恶意软件以及您不记得是否为自己安装过的其他程序。

详细了解如何从计算机中移除恶意软件

面向 Windows 用户的提示：如果您不确定要移除哪些程序，请尝试使用软件清理工具。这是一款应用，可搜索出您计算机上的可疑程序并主动帮您将其移除。

创建新的用户个人资料

如果 Chrome 仍然显示“糟糕！”这一消息，则您的用户个人资料可能已经损坏。

要创建新的用户个人资料，请按以下步骤操作：

  1. 退出 Chrome。
  2. 找到“Default”文件夹，它通常位于以下位置：
    • Windows XP：%USERPROFILE%\Local Settings\Application Data\Google\Chrome\User Data\
    • Windows Vista/7/8：%LOCALAPPDATA%\Google\Chrome\User Data\
    • Mac OX：~/Library/Application Support/Google/Chrome/Default
    • Linux：~/.config/google-chrome/Default
  3. 将该文件夹重命名为“Old Default”。您的旧用户个人资料会保存在此文件夹中，但将不再有效。
  4. 重新启动 Chrome。
