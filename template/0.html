真是遗憾，Google Chrome 不适合您。卸载保平安。

Windows XP

Google Chrome 的删除方式与其他 Windows 程序相同。请按以下步骤操作：

  1. 关闭所有 Chrome 窗口和标签页。
  2. 依次打开开始菜单 > 控制面板。
  3. 点击添加或删除程序。
  4. 双击 Google Chrome。
  5. 点击删除。
  6. 在确认对话框中点击卸载。如果您想删除自己的用户个人资料信息（例如您的浏览器偏好设置、书签和历史记录），请选中“同时删除浏览数据”复选框。

Windows Vista/Windows 7/Windows 8

  1. 关闭所有 Chrome 窗口和标签页。
  2. 依次打开开始菜单 > 控制面板（Windows 8 用户：了解如何访问“控制面板”）。
  3. 点击程序和功能。
  4. 双击 Google Chrome。
  5. 在确认对话框中点击卸载。如果您想删除自己的用户个人资料信息（例如您的浏览器偏好设置、书签和历史记录），请选中“同时删除浏览数据”复选框。

您的 Chrome 是计算机中其他软件附带安装的吗？如果是，那么此版本的 Google Chrome 可能已安装在整个系统中，您需要在您的计算机上登录具有管理员权限的用户帐户，才能卸载该浏览器。如果 Google Chrome 是针对系统中的所有用户安装的，那么卸载该浏览器时会从计算机的所有用户帐户中都删除该程序。

Mac

  1. 转至包含 Google Chrome 的文件夹。
  2. 将 Google Chrome 拖到 Dock 中的垃圾桶图标上。如果 Chrome 安装在“应用”文件夹中，那么您需要拥有管理员权限才能将其删除。要从 Dock 中删除 Chrome，只需用鼠标将其拖出 Dock 即可。

Linux

  1. 依次转至应用 > Debian > 应用 > 系统 > 软件包管理 > 新立得软件包管理器。
  2. 在软件包列表中搜索“Google Chrome”。
  3. 选中“Chrome 软件包”复选框。
  4. 要彻底卸载（包含配置文件），请选中“标记以便彻底删除”。要在卸载时保存某些偏好设置，请选中“标记以便删除”。
  5. 点击应用。
  6. 选中“要彻底删除”菜单中的 Chrome。
  7. 点击应用。
